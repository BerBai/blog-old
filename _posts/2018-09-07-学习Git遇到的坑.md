## 学习Git遇到的坑
- - - - -
## 1. `git commit` 遇到 `error：pathspec`

原因：输入 `git commit -m "messages"` 时，忘记加 `-u` 。

## 2. `git commit -am "messages"` ，不能提交新增文件

先使用 `git add -A` ，提交全部新增文件，再使用 `git commit -m "messages"` 。
